# Cortex Global Coming Soon Page

Static coming-soon landing page for Cortex Global.

## Local Preview

Run a local static server from the project root:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://127.0.0.1:8080/
```

## Files

- `index.html` - page markup, styles, SEO metadata, and Unicorn Studio embed.
- `og.png` - Open Graph/Twitter social preview image.
- `favicon.ico` - browser favicon.
- `mark.svg` - standalone mark used on the page.
- `font/mona-sans.ttf` - self-hosted Mona Sans font.
- `font/OFL.txt` - font license.

## Deployment

This is a static site, so it can be deployed to Vercel, Netlify, Cloudflare Pages, GitHub Pages, or any static hosting provider.
